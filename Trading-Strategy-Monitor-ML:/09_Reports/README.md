# 09_Reports — Executive Reports

## What goes here

Reports and documentation for internal review and project presentation.

These are not the final deliverables (those go to `10_Deliverables`) — these are analytical narratives, draft reports, and presentation notes.

## Typical files

```
09_Reports/
├── phase6_executive_summary.md  → Executive summary of Phase 6 (capital plan)
├── system_overview.md           → Full system presentation
└── backtest_results.md          → Backtest results narrative
```

## Standard report structure

1. Executive Summary (3 lines max)
2. Dataset context and methodology
3. Audit findings
4. Quantitative metrics and analysis
5. Capital management decision with justification
6. Monitoring plan
7. Risk flags and considerations
