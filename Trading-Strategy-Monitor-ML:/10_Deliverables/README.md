# 10_Deliverables — Final Outputs

## What goes here

Final outputs ready for presentation or distribution. All files here must be reviewed and approved before being placed in this folder.

**Rule:** Nothing goes here that hasn't been reviewed and signed off.

## Typical files

```
10_Deliverables/
├── backtest_report_YYYY-MM-DD.pdf    → Final backtest report (PDF)
├── clean_data_final_YYYY-MM-DD.csv   → Final clean dataset
├── metrics_summary_YYYY-MM-DD.xlsx   → Metrics summary spreadsheet
└── system_presentation_YYYY-MM-DD.md → System overview presentation
```

## Naming convention

Always include the date in the filename:
- `backtest_report_2026-04-15.pdf`
- `metrics_summary_2026-04-15.xlsx`
