# 01_Raw — Original Input Files

## Fundamental Rule

**NEVER modify files in this folder.** These are the original source of truth.

If corrections are needed, apply them in later stages (`03_Normalization` or `05_Clean`) and document every change.

## What belongs here

Files exactly as received from the trading platform export:
- `*.csv` — trade history exports
- `*.txt` — secondary log files (OPEN_CSV, CLOSE_CSV, DAY_SUMMARY schemas)
- Any other raw input file received

## What does NOT belong here

- Modified files of any kind
- Files generated by scripts
- Outputs or derived datasets
